THE node_modules DIRECTORY CONTAINS ALL OUR DEPENDECY FILES.
THE public DIRECTORY HAS FILES SUCH AS INDEX.HTML.
THE src DIRECTORY CONTAINS MOST OF OUR IMPORTANT FILES SUCH AS OUR APP AND INDEX JS FILES AND ALL OTHER COMPONENT FILES.
THE package.json CONTAINS NAMES OF ALL OUR PROJECT DEPENDECIES.
THE package-lock.json CONTAINS A LIST OF ALL DEPENDECIES AND THEIR DEPENDECIES TREE.
THE README.md IS A FILE WHERE WE CAN WRITE SOME INFORMATION ABOUT OUR PROJECT.

We can delete package-lock.json and node_modules if we want to reinstall all our dependecies using the npm install/i command.
